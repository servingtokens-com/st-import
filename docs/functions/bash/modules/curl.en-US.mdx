In our next example we use `curl` in cosequence with `eval` to scrape text from a webpage and build a text file called `text-file.txt`. We then write the file to the system's `/tmp` directory.

____________________________________________________________


```bash
#!/usr/bin/env import

# environmental variables for the build
IMPORT_SERVER="https://www.static.servingtokens.com"
# We define our `/tmp` directory to cache the incoming steghide install.
IMPORT_CACHE="/tmp"
# Next, we call for a fresh download of script files (this is like Shift + Reload in the browser).
IMPORT_RELOAD=1
# Lastly, we can log information related to `import` to standard error "a.k.a. stderr".
IMPORT_DEBUG=1

handler() {
    eval "$(curl -LfsS https://static.servingtokens.com >  $IMPORT_CACHE/text-file.txt)" \
    echo "Hello script at: $(cat $IMPORT_CACHE/text-file.txt)"
}
```
